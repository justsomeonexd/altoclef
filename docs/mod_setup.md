# Mod Setup

This project uses the Fabric mod loader. To run the bots, install Fabric and drop the required jars into your Minecraft `mods` folder.

1. **Install Fabric Loader** – [fabricmc.net/use](https://fabricmc.net/use).
2. **Download AltoClef and Baritone JARs** – grab the latest release jars from the GitHub releases page.
3. **Copy JARs** – place both jars inside your `.minecraft/mods` directory.

After installing, you can build and launch clients using the standard Gradle commands, e.g. `./gradlew build -x test` and `./gradlew runClient`.
